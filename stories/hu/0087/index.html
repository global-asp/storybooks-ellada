Επίπεδο 1

Ήχος ιστορίας ακόμη δεν είναι διαθέσιμος.

Szeretnék olvasni.

Μου αρέσει να διαβάζω.

De kinek olvassak?

Σε ποιόν μπορώ να διαβάσω;

A húgom alszik.

Η αδερφή μου κοιμάται.

Kinek olvassak?

Σε ποιόν μπορώ να διαβάσω;

Anyukám és a nagymamám nem érnek rá.

Η μητέρα μου και η γιαγιά μου είναι απασχολημένοι.

Kinek olvassak?

Σε ποιόν μπορώ να διαβάσω;

Apukám és a nagypapám nem érnek rá.

Ο πατέρας μου και ο παππούς μου είναι απασχολημένοι.

Kinek olvassak? Olvasok inkább magamnak!

Σε ποιόν μπορώ να διαβάσω; Μπορώ να διαβάσω στον εαυτό μου!
